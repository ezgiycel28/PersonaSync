# PersonaSync: Kişilik ve Öğrenme Stili Envanteri Modülü

Bu modül, kullanıcıları verimlilik alışkanlıkları ve öğrenme stillerine (VARK) göre analiz eden çekirdek motoru ve veri modellerini içerir.

## Özellikler

*   **OOP Tasarım**: Puanlama ve envanter mantığı temiz sınıflarla yönetilir.
*   **Tip Güvenliği**: Pydantic modelleri ile veri doğruluğu ve süreç takibi.
*   **Puanlama Algoritması**: Hangi cevabın hangi kategoriye (Görsel Öğrenici, Yüksek Erteleme, vb.) etki ettiğini esnek, ağırlıklı puan sistemiyle hesaplar.

## Kurulum

Gerekli bağımlılıkları yükleyin:

```bash
pip install -r requirements.txt
```

## Kullanım

Demo senaryosunu çalıştırmak için:

```bash
python main.py
```

## Testler

Testleri çalıştırmak için:

```bash
pytest
```
